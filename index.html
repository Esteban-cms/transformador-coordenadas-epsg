Transformador de Coordenadas - EPSG

Aplicación en Python para transformar coordenadas entre sistemas EPSG, copiar, eliminar, visualizar en mapa y exportar a Shapefile.

Funciones principales

  • ✔ Transformación entre EPSG
  • ✔ Copia rápida en formato QGIS
  • ✔ Clic derecho para copiar o eliminar coordenadas
  • ✔ Visualización en mapa
  • ✔ Exportación a SHP (EPSG:9377)

Instalación

python -m pip install pandas pyproj folium geopandas shapely openpyxl

Ejecución

python app_transformador.py
